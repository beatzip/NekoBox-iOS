NekoBox-iOS is a lightweight iOS client integrating **sing-box** routing, DNS management, and outbound configuration, designed for privacy-focused proxy setups.

### Features
- Dynamic `config.json` generation
- Shared App Group config for VPN extension
- Modular route and DNS management
- Built-in logging with `NB4ALogger`

### Build Instructions
1. Clone this repo
2. Open `NekoBox-iOS.xcodeproj` or use SwiftPM
3. Build and run the app
