# NekoBox iOS (Demo MVP)

Demo Swift project for NB4A Route + DNS handling

### Features
- Fully typed Swift model for route rules
- JSON encode/decode with ConfigManager
- Logging via NB4ALogger
- Example configuration (`config.json`)
- Ready for GitHub release

### Requirements
- macOS 14+
- Xcode 16+
- Swift 5.9+
